# Heal & Save

Heal & Save is the website for our Informatics 172 class project on HSA literacy. It is aimed at young adults who may feel unsure or stressed about health savings accounts, medical costs, and where to start.

The site introduces the project, explains the team and purpose, links to outside resources, and includes `Savvy`, our embedded HSA chatbot.

## Project focus

This project was built to make HSA information easier to approach.

- `Home` introduces the project and links to the main sections.
- `About Us` explains the team and the motivation behind the project.
- `Savvy` embeds the chatbot experience.
- `Resources` collects references, surveys, and supporting links.

## Tech stack

- `React`
- `TypeScript`
- `Vite`
- `Tailwind CSS`

## Run locally

```bash
npm install
npm run dev
```

## Build for production

